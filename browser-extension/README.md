# 图片搜索与二维码解析器

一个Chrome浏览器扩展，支持右键搜索图片和解析二维码功能。

## 功能特性

### 🔍 图片搜索
- 右键点击任意网页图片
- 支持多个搜索引擎：Google、百度、Yandex、TinEye等
- 可自定义添加搜索引擎
- 一键搜索，新标签页打开结果

### 📱 二维码解析
- 右键点击包含二维码的图片
- 自动识别和解析二维码内容
- 支持文本复制和链接跳转
- 结果悬浮显示，用户友好

### ⚙️ 搜索引擎管理
- 内置常用搜索引擎模板
- 支持启用/禁用搜索引擎
- 可编辑和删除自定义搜索引擎
- 简单直观的设置界面

## 安装方法

### 开发者模式安装
1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本扩展的文件夹

### 使用说明

#### 图片搜索
1. 在任何网页上找到您想搜索的图片
2. 右键点击图片
3. 选择"搜索图片" → 选择搜索引擎
4. 新标签页中查看搜索结果

#### 二维码解析
1. 在网页上找到包含二维码的图片
2. 右键点击图片
3. 选择"解析二维码"
4. 页面右上角显示解析结果

#### 管理搜索引擎
1. 点击扩展图标打开弹窗
2. 点击"搜索引擎设置"
3. 添加、编辑或删除搜索引擎
4. 使用预设模板快速添加常用引擎

## 支持的搜索引擎

### 内置搜索引擎
- **Google 图片搜索**: 全球最大的图片搜索引擎
- **百度识图**: 中文图片搜索，适合中文内容
- **Yandex 图片搜索**: 俄语搜索引擎，图片识别能力强
- **TinEye 反向搜索**: 专业的反向图片搜索
- **Bing 图片搜索**: 微软的图片搜索服务

### 自定义搜索引擎
可以添加任何支持图片URL参数的搜索引擎，URL格式示例：
```
https://www.google.com/searchbyimage?image_url=
```

## 技术实现

- **Manifest V3**: 使用最新的Chrome扩展API
- **Context Menus API**: 实现右键菜单功能
- **Storage API**: 保存用户设置和搜索引擎配置
- **JavaScript 图像处理**: 本地二维码解析
- **现代化UI**: 响应式设计，用户体验良好

## 隐私说明

- 本扩展不收集任何用户数据
- 图片搜索通过跳转到第三方搜索引擎进行
- 二维码解析在本地进行，不上传图片到服务器
- 仅需要必要的浏览器权限：右键菜单、存储、活动标签页
