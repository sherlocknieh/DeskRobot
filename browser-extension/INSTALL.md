# 浏览器扩展安装和使用指南

## 快速安装

### Chrome/Edge 浏览器
1. 打开浏览器，访问 `chrome://extensions/` (Chrome) 或 `edge://extensions/` (Edge)
2. 开启右上角的"开发者模式"开关
3. 点击"加载已解压的扩展程序"按钮
4. 选择 `browser-extension` 文件夹
5. 扩展安装完成，会在浏览器工具栏显示图标

## 功能使用

### 图片搜索功能
1. **浏览任意网页**，找到您想要搜索的图片
2. **右键点击图片**，会出现上下文菜单
3. **选择"搜索图片"**，展开搜索引擎子菜单
4. **选择搜索引擎**（如Google、百度等）
5. **新标签页自动打开**，显示搜索结果

### 二维码解析功能
1. **找到包含二维码的图片**（可以是网页上的任意二维码图片）
2. **右键点击图片**
3. **选择"解析二维码"**
4. **页面右上角显示解析结果**
   - 如果是URL链接，可以直接点击访问
   - 如果是文本，可以复制到剪贴板
   - 结果框5秒后自动消失，也可手动关闭

### 搜索引擎管理
1. **点击扩展图标**（浏览器工具栏中的图标）
2. **查看当前配置**：弹窗显示已启用的搜索引擎
3. **打开设置页面**：点击"搜索引擎设置"按钮
4. **管理搜索引擎**：
   - 添加新搜索引擎：输入名称和URL
   - 启用/禁用：使用开关控制
   - 编辑：修改名称或URL
   - 删除：移除不需要的搜索引擎
5. **使用模板**：点击预设按钮快速添加常用搜索引擎

## 搜索引擎URL格式

添加自定义搜索引擎时，URL格式为：
```
搜索网站的图片搜索地址 + 图片URL参数位置
```

### 示例：
- **Google**: `https://www.google.com/searchbyimage?image_url=`
- **百度**: `https://graph.baidu.com/s?sign=&rt=upload&rn=10&ct=1&tn=baiduimage&objurl=`
- **Yandex**: `https://yandex.com/images/search?rpt=imageview&url=`

## 故障排除

### 右键菜单不显示
- 检查扩展是否正确安装并启用
- 刷新当前网页
- 确保右键点击的是图片而不是其他元素

### 二维码解析失败
- 确保图片清晰，二维码完整
- 当前版本使用简化算法，复杂二维码可能无法识别
- 建议使用高质量、对比度清晰的二维码图片

### 搜索结果无法打开
- 检查网络连接
- 某些搜索引擎可能在特定地区不可用
- 尝试使用其他搜索引擎

### 设置页面无法打开
- 右键点击扩展图标，选择"选项"
- 或访问 `chrome://extensions/`，找到本扩展，点击"详细信息" → "扩展程序选项"

## 隐私和安全

- ✅ 本扩展不收集任何个人数据
- ✅ 图片搜索通过跳转到第三方搜索引擎进行
- ✅ 二维码解析在本地进行，不上传到服务器
- ✅ 所有设置保存在本地浏览器中

## 支持的浏览器

- Chrome 88+
- Microsoft Edge 88+
- 其他基于Chromium的浏览器

## 获取帮助

如果遇到问题：
1. 检查浏览器控制台是否有错误信息
2. 尝试禁用其他可能冲突的扩展
3. 重新安装扩展
4. 确保浏览器版本符合要求
